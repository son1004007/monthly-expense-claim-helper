# 근태 메모 입력 예시

실제 근태 메모는 `local/attendance/2026-06.md` 또는 `attendance-notes/2026-06.md`에 작성하고 Git에 커밋하지 않는다.

## 입력 규칙

- 한 줄에 하루의 근태 정보를 적는다.
- 날짜는 `5일`, `2026-06-05`, `6/5` 형식을 허용한다.
- 야근식대 상세내역에는 퇴근 시간을 사용한다.
- 외근 교통비 상세내역에는 교통사용 내역의 승차/하차 내용을 사용하고, 근태 메모의 외근지를 검증 보조 정보로 사용한다.
- 연차/반차일의 교통 내역은 `review_required`로 표시한다.
- 주말/공휴일 대중교통 내역은 근태 메모에 `주말출근`, `휴일근무`, `외근` 등 출근 근거가 있는 날짜만 청구 후보로 둔다.
- 주말/공휴일인데 출근 근거가 없으면 `exception_weekend_no_attendance`로 표시하고 웹브라우저에는 입력하지 않는다.

## 예시

```text
1일 20시30분
2일 연차
3일 외근 고객사A
4일 외근 고객사A
5일 오전 반차
6일 외근 고객사B
7일 21시10분
8일 22시00분
9일 외근 고객사C
10일 외근 고객사C
13일 주말출근 고객사D
14일 휴일근무 20시20분
```
